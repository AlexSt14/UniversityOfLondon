Introduction:

My website tells the story of a local and very talented make up artist in my area. As requested I have contacted a local business, partnered with the business to discuss their needs for this website, exchanging data, receiving the relevant content that I will be using for this website and exchange ideas on the design part of the website. The make up artist is specialized into Everyday Make-Up, Bridal Make-Up, Henna Eyebrows and Waxing Eyebrows At first it was quite a long discussion until we have decided on the structure and the design of the website, as per the layout, the client did not give me clear ideas so I decided I will take ownership and create some layouts that they could choose. The website is structured into 5 different pages, Home, About, Services, Contact and Portfolio. The Homepage is giving an overview of nearly all pages in short while also directs the user to the according page to see the full content relevant for that section. The About page talks about the background and specializations of the make up artist, while the Services page talks about the services that the business will offer, the Contact page contains all the details required to get in contact with the business and finally, the Portfolio page contains a broad overview of pictures of a lot of work that the make up artist did. The visual design is a modern one with a mix of warm colors with a dark grey, while the HTML part of the website is done quite a lot with Bootstrap but heavily modified for my client's needs.

Inspiration:

I was inspired by the make up artist itself while discussing the needs of the client and explaining me what and how would they want it. I also got inspiration from other websites that were created for make up artists, lastly I was inspired by different artists around the web. To summarize, I have been inspired by:

  • The client itself
  • Other Make-Up Artists Websites
  • Different other artists

Accessibility:

My website is accessible through the following ways:

  • Focus indication for interactive elements like links and buttons
  • Used correct Form labels with their respective fields
  • Proper use of heading tags to create hierarchy

Usability:

As for usability, I considered the following ways on my website:

  • Semantic HTML where appropriate and possible
  • Alt attributes to images and error messages for videos in case of fail load
  • Color contrast between foreground and background colors, especially for text. This helps visually impaired users

Learning:

The first part that I had to learn was to set up a slideshow with Javascript. I learned how to do that by searching on google (This is also documented in the code of my website).

The second part that I needed to learn was the use of @media for the website to be responsive. I have learned this during the lessons of this module.

The third part that I learned and it was the most important one was how the Bootstrap grid works. I had to understand how the grid works before starting to work on the website. To learn this I have used the main Bootstrap website which will be detailed down below in resources.

Evaluation 1:

The design I believe is quite well done and close to commercial ready websites. The flow of the website and the colors chosen match the idea of the website and having sent the website to the client I was told the design was through the best they have seen. They specifically loved the Homepage, Services and Portfolio page.

Evaluation 2:

One important aspect that I could improve is the sizing of the images on the Homepage and Portfolio page at the end of the page. Because images are not all the same size and because I use the Masonry CDN to arrange the cards dynamically, I have a gap between the footer section and the images themselves. I can and will improve this by taking each image at a time and process it to make all images the same size in order to eliminate that gap between the footer and the images.
